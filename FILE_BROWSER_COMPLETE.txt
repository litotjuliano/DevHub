  HOW TO USE (QUICK START)
================================================================================

1. Open DevHub Dashboard
   http://localhost:8080

2. Find Resource & Docs Section
   Look for cyan buttons

3. Click "📚 Browse All Docs"
   Modal opens showing folder tree

4. Expand Folders
   Click ▶ arrow next to folder name
   Arrow changes to ▼

5. Click File to View
   📄 FILENAME.md
   Content appears on right panel

6. Read Markdown Content
   - Headers formatted
   - Code highlighted
   - Links clickable
   - Scroll to read all

7. Navigate Between Files
   - Click different files
   - Content updates instantly

8. Close When Done
   Click ✕ button or press Escape
