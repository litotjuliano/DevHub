📚 CHEATSHEET: Reading Test Reports

📄 Read Report with Bash
cat test-report-2026-05-30-10-00-00.json
Replace filename with your downloaded report
❌ Find Failed Tests
grep '"completed": false' test-report-*.json
Shows all incomplete/failed tests
📊 Extract Summary
grep -A 4 '"summary"' test-report-*.json
Shows completion stats
🔍 Find Specific Issue
grep -i "error\|fail\|bug" test-report-*.json
Search for keywords in notes
📈 Count by Status
grep -c '"completed": true' test-report-*.json
Count passed tests
🔄 Compare Reports
diff report1.json report2.json
See what changed between reports
💡 Quick Reference
✅ Passed Tests

❌ Failed Tests
